# 更新日志

所有重要的更新都将记录在此文件中。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2026-05-12

### 🎉 初始发布

首个可用版本，包含完整的3D太阳系探索体验。

### ✨ 新增

- **3D 太阳系场景**
  - 基于 NASA JPL 数据的8大行星 + 太阳真实轨道
  - 牛顿迭代法求解开普勒方程的精确椭圆轨道
  - 递归卫星系统渲染（含月球、伽利略卫星等）
  - 小行星带可视化
  - 真实比例 / 夸张比例双模式切换

- **交互式教育模块**（20+）
  - 🎯 探索任务系统 — 引导式太阳系探索
  - 🔮 预测挑战 — 预测行星位置竞赛
  - 🧪 沙盘实验 — 实时调整轨道参数
  - 🚀 霍曼转移轨道设计器 — 可视化轨道转移设计
  - 🌑 月食实验 — 日月食几何模拟
  - 🕳️ 黑洞模拟器 — 引力透镜与事件视界
  - ⚡ 光速旅程模式 — 带时间膨胀HUD的星际旅行
  - 📝 知识测验 — 自适应难度题库
  - 📊 跨学科面板 — 天文与多学科关联
  - 🔭 观测指南 — 实用观星建议
  - 🛰️ 航天器信息面板
  - 🔬 科学前沿资讯
  - 🗳️ 科学争议投票

- **游戏化系统**
  - 30+ 成就系统，追踪探索进度
  - 成就解锁动画与音效
  - 故事任务进度追踪
  - 知识点分级解锁（青铜/白银/黄金）

- **HUD 与视觉**
  - 科幻风格毛玻璃面板
  - 文字发光效果
  - 动态角落装饰线框
  - 加载画面与欢迎引导
  - 响应式布局适配

- **音频系统**
  - Web Audio API 原生音效
  - 环境氛围音
  - 交互反馈音效
  - 音频上下文自动恢复处理

- **技术特性**
  - React 18 + TypeScript 严格模式
  - Three.js + React Three Fiber 3D渲染
  - Zustand 全局状态管理
  - Tailwind CSS 科幻主题
  - Framer Motion 动画
  - 纯静态站点，零后端依赖
