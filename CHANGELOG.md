## [0.2.0] - 2026-05-13

### 🎯 天文学家教育审核改进

由专业天文学家和教育学家进行的深度审核后，实施了系统性教育改进，从"炫技展示"转向"育人导向"。

### ✨ 新增

- **教育内容扩展**
  - 补全5大行星的Bronze级科普知识（火星、金星、水星、天王星、海王星），每个含Bronze/Silver/Gold三级+追问链
  - Silver/Gold级知识增加"桥梁文案"过渡，从具体比喻平滑过渡到抽象数学
  - 跨学科连接追加💡思考标签，引导学生主动建立跨学科推理
  - 测验题库新增科学史误解题型（洛威尔火星运河、冥王星降级、科学修正过程）

- **成就系统教育化改造**
  - 18个成就解锁Toast从"收集导向"改为"反思导向"，每个成就附带认知钩子反问
  - 降低`bronze_scholar`解锁门槛（5→3），让更多学生能体验Silver/Gold级内容

- **比例尺感受器**
  - 新增"如果太阳是篮球"类比面板，以操场尺度直观展示太阳系的真实空旷
  - 各行星的实际尺寸（毫米/厘米/米）和相对距离一目了然

- **天文学家画廊**
  - 新增8位多元背景天文学家的档案（5位女性，3位非西方文化背景）
  - 涵盖：张衡、阿尔·花拉子米、甘德与石申、亨丽爱塔·勒维特、约瑟琳·贝尔、薇拉·鲁宾、玛丽亚·米切尔、玛格丽特·伯比奇
  - 每位含生平、贡献、趣闻、搜索关键词，引导学生继续探索

- **黑洞模拟器增强**
  - 新增黑洞质量滑块（1x~1000x太阳质量），让学生对比恒星质量与超大质量黑洞的潮汐力差异
  - 状态阈值随质量自适应，超大质量黑洞事件视界处不再"必死"
  - 强化教育说明：区分不同质量黑洞的物理特性

- **四季星空面板**
  - 北半球四季主要观测目标指南，含行星、星座、深空天体
  - 简化方位示意图，指导学生实际观星方向

- **探索报告导出**
  - 一键生成Markdown格式学习报告，含已探索天体、成就、推进时间
  - 附科学态度宣言，可作为学校科学课实践作业提交

- **观测指南增强**
  - 新增观测工具分级推荐（肉眼/双筒/望远镜）
  - 推荐4款天文App（Stellarium Mobile、Star Walk 2等）

- **科学前沿教育卡片**
  - 在科学前沿Tab中固定展示"关于本应用的科学模型"教育说明
  - 用水星近日点进动案例解释"模型与现实的裂缝"是科学发现的机会

- **声音物理声明**
  - 欢迎页新增声明：太空中无空气传播声音，音效为交互反馈和数据艺术化呈现

### 🔧 改进

- **真实比例模式**下不可见行星现在显示脉冲信标，避免学生困惑"行星去哪了"
- **知识升级动画**从简单淡入淡出增强为Framer Motion spring弹性动画
- 全部TypeScript代码通过严格模式检查，零类型错误

## [0.1.0] - 2026-05-12

### 🎉 初始发布

首个可用版本，包含完整的3D太阳系探索体验。

### ✨ 新增

- **3D 太阳系场景**
  - 基于 NASA JPL 数据的8大行星 + 太阳真实轨道
  - 牛顿迭代法求解开普勒方程的精确椭圆轨道
  - 递归卫星系统渲染（含月球、伽利略卫星等）
  - 小行星带可视化
  - 真实比例 / 夸张比例双模式切换

- **交互式教育模块**（20+）
  - 🎯 探索任务系统 — 引导式太阳系探索
  - 🔮 预测挑战 — 预测行星位置竞赛
  - 🧪 沙盘实验 — 实时调整轨道参数
  - 🚀 霍曼转移轨道设计器 — 可视化轨道转移设计
  - 🌑 月食实验 — 日月食几何模拟
  - 🕳️ 黑洞模拟器 — 引力透镜与事件视界
  - ⚡ 光速旅程模式 — 带时间膨胀HUD的星际旅行
  - 📝 知识测验 — 自适应难度题库
  - 📊 跨学科面板 — 天文与多学科关联
  - 🔭 观测指南 — 实用观星建议
  - 🛰️ 航天器信息面板
  - 🔬 科学前沿资讯
  - 🗳️ 科学争议投票

- **游戏化系统**
  - 30+ 成就系统，追踪探索进度
  - 成就解锁动画与音效
  - 故事任务进度追踪
  - 知识点分级解锁（青铜/白银/黄金）

- **HUD 与视觉**
  - 科幻风格毛玻璃面板
  - 文字发光效果
  - 动态角落装饰线框
  - 加载画面与欢迎引导
  - 响应式布局适配

- **音频系统**
  - Web Audio API 原生音效
  - 环境氛围音
  - 交互反馈音效
  - 音频上下文自动恢复处理

- **技术特性**
  - React 18 + TypeScript 严格模式
  - Three.js + React Three Fiber 3D渲染
  - Zustand 全局状态管理
  - Tailwind CSS 科幻主题
  - Framer Motion 动画
  - 纯静态站点，零后端依赖
